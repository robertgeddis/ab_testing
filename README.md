# A/B Testing 

Comprehensive selection of code providing an end-to-end framework for A/B testing. 
These scripts provided the architecture to design, monitor and analyse tests that drove decisions on new features, marketing approaches and pricing strategies. 

## 📈 Focus Areas
* **Pricing Elasticity:** Analyzing user behavior and conversion rates across different price points.
* **Profile Ranking:** Measuring the impact of changes to the platform's matching algorithm.
* **Landing Page Optimization:** Iterating over multi-channel landing pages to improve funnel conversion.

## 🛠 Technical Features
* **Testcell Assignment:** Ensuring users are correctly attributed to 'Control' vs 'Test' groups at the point of first interaction.
* **Metric Standardization:** A unified matrix to compare primary KPIs and secondary guardrail metrics across all tests.
* **Statistical Rigor:** Logic designed to output raw data structured for significance and confidence in downstream tools like Tableau and Amplitude.

## 💼 Business Impact
* **Reduce Risk:** Validate changes to the user experience and pricing stratgey before a monitored gradual global roll out.  
* **Quantify Value:** Set expectations and better understand the cost benefit of undertaking extensive work.
* **Iterate Faster:** Standardized reporting reduced the time-to-insight and enabled for a constant cycle of experimentation.
